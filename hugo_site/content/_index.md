# Paper Insight 论文情报站

**AI 驱动的多行业论文情报站**，自动抓取和分析全球最新学术论文和开源项目。

## 支持行业

- AI/ML - 机器学习、深度学习、大语言模型
- 医疗健康 - 医学前沿、药物研发
- 金融 - 量化交易、区块链
- 新能源/材料 - 太阳能、电池、氢能
- 半导体/芯片 - IC设计、半导体工艺
- 消费/电商 - 推荐系统、供应链
- 游戏 - 图形学、AI游戏

## 数据来源

| 来源 | 覆盖领域 |
|------|----------|
| arXiv | AI/ML、新能源、半导体 |
| PubMed | 医疗健康 |
| Semantic Scholar | 金融、消费、游戏 |
| GitHub Trending | 开源项目趋势 |

## 快速开始

```bash
# 单次运行
./start.sh once

# 持续运行
./start.sh continuous
```

## Docker 部署

```bash
docker-compose up -d
```

访问 http://localhost:8084
