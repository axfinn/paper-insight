# Paper Insight

**让 AI 为你每天追踪全球最新论文和开源项目**

---

Paper Insight 是一款 AI 驱动的情报自动化工具，专注于追踪 AI/ML、医疗健康、金融、新能源等 7 大前沿领域的最新学术论文和 GitHub 开源项目。

每天自动抓取 → AI 深度分析 → 生成结构化报告和排行榜。

---

## 一句话介绍

> Paper Insight：你的 AI 情报员，7x24 小时追踪全球最新论文和开源项目，自动分析生成有价值的报告和排行榜。

---

## 解决的问题

| 痛点 | 解决方案 |
|------|----------|
| 论文太多，跟不上进展 | AI 自动抓取 + 提炼摘要 |
| 不知道哪些项目值得关注 | GitHub 排行榜按潜力评分排序 |
| 技术调研耗时费力 | 自动生成结构化分析报告 |
| 需要持续跟踪不能一次了事 | 7x24 小时自动运行 |

---

## 核心功能

**1. 多源论文抓取**
- arXiv（AI/ML、新能源、半导体）
- PubMed（医疗健康）
- Semantic Scholar（金融、消费、游戏）

**2. GitHub Trending 追踪**
- Python / Go / Rust / TypeScript / JavaScript
- 按语言分类，潜力评分排序

**3. AI 智能分析**
- 创新点提炼
- 技术栈识别
- 应用场景分析
- 潜力评分（1-5星）

**4. 自动化运行**
- 定时抓取（默认 2 小时）
- 自动清理旧数据
- 支持 AutoDev 自主迭代
